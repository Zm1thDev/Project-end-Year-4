Project_end : STUDENT PROJECT DATA MANAGEMENT SYSTEM FROM PDF FILE

This project develops a student project data management system from PDF files to improve data access and searching efficiency. 
Searching for information from project files is often time-consuming because filenames do not clearly describe their contents. 
The system allows users to upload PDF files, extract important information, and store the project data automatically. 
It is a web application developed using JavaScript and Python with React and Bootstrap, using the Google Drive API for file storage and Google Sheets API for data storage. 
Testing shows that the system can successfully extract key information from PDF files, helping users review content quickly and reduce manual search time.

Tech : React, Bootstrap, FastAPI, Python, PyPDF2, Regular Expression, Drive API, Sheet API
